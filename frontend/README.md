# Soterra Frontend

Next.js frontend for the Soterra product UI.

## Local development

1. Install dependencies:
   `npm install`
2. Set `BACKEND_BASE_URL` to the public or local URL of the backend service.
3. Start the app:
   `npm run dev`

## Deploy to Vercel

- Root directory: `frontend`
- Framework preset: Next.js
- Required environment variable:
  - `BACKEND_BASE_URL`
