# Soterra Database

Supabase schema and migration files for the Soterra backend.

## Contents

- `supabase/migrations/`
  SQL migrations to apply to your Supabase project

## Recommended usage

1. Create a Supabase project.
2. Apply the SQL in `supabase/migrations`.
3. Create the storage bucket expected by the backend.
4. Copy the Supabase URL and service role key into the backend deployment environment.
